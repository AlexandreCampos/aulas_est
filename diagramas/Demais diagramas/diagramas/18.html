Produzir relatório financeiro de receita de aluguéis
UseCase
Breve Descrição Este caso de uso descreve o fluxo de eventos que ocorre durante o processo de produção de relatório financeiro de receita de aluguéis da locadora. Atores Envolvidos Funcionário Pré-condições O funcionário precisa estar cadastrado e autenticado no sistema. Fluxo Básico de Eventos 1. O funcionário acessa uma tela com diversos relaórios financeiros disponíveis e suas descrições. 2. O funcionário seleciona o relatório financeiro de receita de alugéis e coloca em execução. 3. O sistema executa o relatório financeiro. Ou seja, acessa os registros de alugueis de todos os clientes e soma os valores de receita. Este processo pode demorar alguns minutos. O status fica 'em execução'. 4. O sistema termina a execução do relatório. O status é atualizado para 'concluído'. 5. O sistema gera um relatório em um documento csv para ser baixado. 6. O funcionário clica em download para receber o documento e trabalhar com os dados. Fluxos Alternativos No passo 3, o sistema ocorre algum erro no processamento de criação relatório financeiro. O status é atualizado para 'erro'. Uma mensagem com os detalhes do erro é exibida na tela. O funcionário pode executar novamente ou tentar mais tarde. Pós-condições O relatório financeiro é gerado no sistema.
