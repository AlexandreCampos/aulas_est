Alugar veículo
UseCase
Este caso de uso descreve o fluxo de eventos que ocorre durante o processo de locação de veículos na loja da locadora. Atores Envolvidos Cliente, Atendente Pré-condições O cliente precisa estar cadastrado e autenticado no sistema Fluxo Básico de Eventos 1. O Cliente escolhe o local de retirada do veículo. 2. O Cliente seleciona o período da locação (data inicial e final). 3. O sistema verifica a disponibilidade de veículos para o local e período selecionados. 4. O sistema apresenta os veículos disponíveis. 5. O Cliente seleciona o veículo desejado. 6. O sistema calcula do valor da locação e apresenta para o cliente. 7. O Cliente confirma a reserva da locação. 8. O sistema solicita a forma de pagamento (débito, crédito ou boleto bancário). 9. O cliente entra com a forma de pagamento. 10. O sistema registra a forma de pagamento na reserva do veículo. 11. O sistema fecha a reserva e a apresenta as informações para o cliente. 12. O caso de uso termina. Fluxos Alternativos No passo 3, o sistema verifica que não existem veículos disponíveis na loja selecionado pelo cliente. Neste caso, o sistema apresenta uma mensagem para o cliente informando a indisponibilidade e o caso de uso termina. Pós-condições A reserva de locação é registrada no sistema.
