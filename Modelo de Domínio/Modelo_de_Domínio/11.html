Atualizar dados de veículos
UseCase
Breve Descrição Este caso de uso descreve o fluxo de eventos que ocorre durante o processo de atualização dados de um veiculo da locadora. Pré-condições O funcionário precisa estar cadastrado e autenticado no sistema. Legenda para o texto abaixo: A: Ator S: Sistema Fluxo Básico de Eventos 1. A - O funcionário informa qual veículo deseja atualizar dados. 2. S - O sistema faz a busca do veículo. 3. S - O sistema apresenta os detalhes do veículo, caso exista. 4. A - O funcionário seleciona o formulário de editar informações do veículo. 5. A - O funcionário atualiza as informações necessárias. 6. S - O sistema valida se as atualizações estão nos formatos corretos e dentro das condições estabelecidas. 7. A - O sistema informa que as atualizações foram realizadas com sucesso. 8. S - O caso de uso termina. Fluxos Alternativos No passo 6, o sistema verifica que algum dado não estava com os formatos corretos e dentro das condições estabelecidas. Neste caso, o sistema apresenta uma mensagem para o funcionário solicitando que atualize novamente o dado incorreto e quais são as regras. Pós-condições Os dados do veículos são atualizados no sistema.
