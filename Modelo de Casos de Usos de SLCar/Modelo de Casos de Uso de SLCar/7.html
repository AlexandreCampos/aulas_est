Produzir relatórios financeiros
UseCase
Breve Descrição Este caso de uso descreve o fluxo de eventos que ocorre durante o processo de produção de relatório financeiro da locadora. Atores Envolvidos Funcionário Pré-condições O funcionário precisa estar cadastrado e autenticado no sistema. Legenda para o texto abaixo: A: Ator S: Sistema Fluxo Básico de Eventos 1. A - O funcionário acessa uma tela com diversos relaórios financeiros disponíveis e suas descrições. 2. A - O funcionário seleciona um relatório financeiro e coloca em execução. 3. S - O sistema executa o relatório financeiro. Dependendo do tipo de relatório, pode demorar alguns minutos. O status fica 'em execução'. 4. S - O sistema termina a execução do relatório financeiro. O status é atualizado para 'concluído'. 5. S - O sistema gera um relatório em um documento para ser baixado. 6. A - O funcionário clica em download para receber o documento e trabalhar com os dados.
